Details about project

Fertilizer prediction:

This Web interface is software part of our Fertilizer prediction project. It also works independentely as you can enter details in prediction page and click on Get Result button to fetch prediction results.

Working :

At backend a python code is handling user queries and Machine learning part.

JSONify :

if you want to implement this project and want to use our predcition model you can use JSON based API. Python code for Fetching data through API is:

# filename = 'predict.py'
import requests
url = 'https://fertlmodel.herokuapp.com/api'
#r = requests.post(url,json={'V1':Temperature, 'V2':Humidity, 'V3':Moisture, 'V4':Soil_Type, 'V5':Crop_Type, 'V6':Nitrogen, 'V7':Potassium, 'V8':Phosphorus,})
#In API dictionary datatype for keys are of string datatype and values datatype is int as shown in example below:
r = requests.post(url,json={'V1':27, 'V2':55, 'V3':54, 'V4':1, 'V5':6, 'V6':12, 'V7':4, 'V8':4,})
# Get result back in json and use below code to get string output.
print(r.json())
